Plum Blossom I Ching

Longterm analysis and trends

Analysis of circumstance which is out of our limited sight. It is used for longterm predictions, for instance when we need to know future of the family, agricultural or social situtation of the globe. It can be used for prediction of tendency in marriage, business or any kind of relationship.

Using this method, it is possible to predict political development or progress of longterm education.

Usage:

  • Analysis of name
  • Analysis of place
  • Name analysis in connection to place
  • Name analysis in connection to job, education, partner, place of living, company, etc.
  • Family constellation
